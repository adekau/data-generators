Const sequence

  • sequence<T>(...generators: DataGenerator<T>[]): DataGenerator<undefined | T>
    • Defined in creation/sequence.ts:18

    Creates a sequence of generators that on each create call creates from the next generator. This includes createMany where each output in the array uses the next generator.

    example

// ['r', '6wCg', 'Bob']

Const struct

  • struct<T>(generators: {[ K in string | number | symbol]: DataGenerator<T[K]> }): DataGenerator<T>
    • Defined in creation/struct.ts:23

    Creates a generator that generates an object adhering to an interface using the provided generators for interface members. Can be thought of as behaving similarly to RxJS forkJoin.

    example
    struct({
    isRequired: booleanGenerator(),
    fieldId: integerGenerator(),

}).create();
// Example Output: { isRequired: true, fieldId: 52, value: 'aVE3^x' }
